# PulseList · 轻量生活协作面板

一个面向个人/家庭/小团队的轻量任务与习惯管理应用。核心理念：**低学习成本 + 现代感 UI + 可扩展**。

## 你能做什么
- 任务管理：分类 / 优先级 / 截止日期 / 完成度
- 习惯打卡：连续天数与节奏反馈
- 专注节奏：番茄式专注入口与完成提示
- AI 观察：基于完成度的策略建议（可拓展）

## 运行
```bash
npm install
npm run dev
```

## 规划路线图
1. **MVP**：本地存储 + 今日面板（已完成基础版）
2. **增强**：日历视图 / 推送提醒 / 共享协作
3. **智能**：AI 任务拆解 / 动态优先级 / 成就系统

---

技术栈：Vue 3 + TypeScript + Vite
